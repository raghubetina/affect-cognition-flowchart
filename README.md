# Affect-Cognition Flowchart

An interactive flowchart based on Gerald L. Clore and Alexander J. Schiller's chapter on the affect-cognition connection.

Live site:
<https://raghubetina.github.io/affect-cognition-flowchart/>

The page presents the chapter as a clickable decision tree. Each node opens a modal with a fuller explanation, examples, and supporting textbook page references.

Local preview:

```bash
python3 -m http.server 8765
```

Then open:
<http://127.0.0.1:8765/>
