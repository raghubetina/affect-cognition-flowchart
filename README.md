# Affect-Cognition Flowchart

An interactive flowchart based on Gerald L. Clore and Alexander J. Schiller's chapter "New Light on the Affect-Cognition Connection" in [Handbook of Emotions, Fourth Edition](https://www.guilford.com/books/Handbook-of-Emotions/Barrett-Lewis-Haviland-Jones/9781462536368).

Live site:
<https://raghubetina.github.io/affect-cognition-flowchart/>

The page presents the chapter as a clickable decision tree. Each node opens a modal with a fuller explanation, examples, and supporting textbook page references.

Local preview:

```bash
python3 -m http.server 8765
```
